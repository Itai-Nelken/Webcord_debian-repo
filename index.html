electron-discord-webapp_debian-repo

Unnoficial apt repo for electron-discord-webapp by SpacingBat3

To add this repo to your system, run:

wget -qO- https://raw.githubusercontent.com/Itai-Nelken/electron-discord-webapp_debian-repo/main/add-repo.sh | sudo bash

Alternatively download this file
and move it to /etc/apt/sources.list.d/:

then run this command to add the key: wget -qO- https://raw.githubusercontent.com/Itai-Nelken/electron-discord-webapp_debian-repo/main/KEY.gpg | sudo apt-key add -

Remember to run sudo apt update after doing that.

Now you can install electron-discord-webapp by runningsudo apt install electron-discord-webapp.
