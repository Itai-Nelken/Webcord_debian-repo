WebCord (electron-discord-webapp) debian repo

Unofficial apt repo for WebCord (previously electron-discord-webapp) by SpacingBat3

To add this repo to your system, run:

wget -qO- http://download.tuxfamily.org/webcordapt/add-repo.sh | sudo bash

Alternatively download THIS FILE
and move it to /etc/apt/sources.list.d/:

then run this command to add the key:
wget -qO- http://download.tuxfamily.org/webcordapt/webcord-repo_pub-key.gpg | sudo apt-key add -

Remember to run sudo apt update after doing that.

Now you can install electron-discord-webapp by running sudo apt install webcord .
If you get a 'package not found' error, run sudo apt update and try to install WebCord again.

IMPORTANT! if you have any problems with WebCord installed from this repo, open an issue here and not in the webcord repo unless its a problem with the app itself!
